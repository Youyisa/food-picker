# 🍱 今天吃什么？ (Random Food Picker)

一个专为“选择困难症”患者打造的随机美食选择器。

## ✨ 功能特点

- **🎲 随机选择**：一键点击，快速帮你决定下一顿吃什么。
- **📝 自定义菜单**：
  - 支持添加你喜欢的美食。
  - 支持删除不喜欢的选项。
  - **持久化存储**：你的菜单会自动保存在浏览器中，刷新页面不会丢失。
- **🔄 恢复默认**：一键重置回初始的推荐菜单。
- **🎨 精美界面**：现代化的 UI 设计，带有平滑的动画效果。

## 🚀 如何使用

### 方式一：直接访问（推荐）
访问在线地址https://youyisa.github.io/food-picker

### 方式二：本地运行
1. 下载本项目代码。
2. 直接双击打开 `index.html` 文件即可使用。

## 🛠️ 部署指南 (GitHub Pages)

如果你想把这个网页分享给朋友，可以免费部署到 GitHub Pages：

1. 在 GitHub 上创建一个新的公开仓库（例如 `random-food-picker`）。
2. 将 `index.html`, `style.css`, `script.js` 上传到仓库根目录。
3. 进入仓库 **Settings** -> **Pages**。
4. 在 **Branch** 处选择 `main` 分支并保存。
5. 等待几分钟，GitHub 会生成一个访问链接给你。
